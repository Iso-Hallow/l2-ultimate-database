## 📌 О проекте

**L2 Ultimate Database** — это веб-приложение-шпаргалка для администраторов и игроков серверов Lineage 2. Инструмент помогает быстро находить GM команды, горячие клавиши, ID предметов и координаты.

### 🎯 Основные возможности

| Функция | Описание |
|---------|----------|
| 🤖 **Генератор команд** | Интерактивные поля ввода для команд с аргументами |
| ⭐ **Избранное** | Сохранение часто используемых команд с экспортом/импортом |
| 🔍 **Быстрый поиск** | Фильтрация по названию, команде или описанию |
| 🌓 **Темы оформления** | Тёмная и светлая тема на выбор |
| 🌐 **Локализация** | Русский и английский языки |
| 📱 **Адаптив** | Работает на ПК, планшетах и смартфонах |

## 📱 Особенности работы на планшетах и смартфонах

Проект изначально разрабатывался с учётом мобильного использования. Почему это важно:

| Ситуация | Как помогает |
|----------|--------------|
| Вы не за компьютером, а сервер "летит" | Открыли телефон → нашли команду → скопировали → отправили в чат/коллеге |
| Администрируете с планшета | Кнопки крупные, не мажете мимо, текст читается без зума |
| Быстро нужно подсмотреть ID предмета | Правый сайдбар всегда под рукой, копируется в один клик |
| Поделиться командой в чате | Скопировали готовую команду с аргументами — вставили в Telegram/Discord |

**Что адаптировано:**
- ✅ Сетка команд превращается в одну колонку на узких экранах
- ✅ Кнопки копирования достаточно крупные для пальцев
- ✅ Панель категорий скроллится горизонтально (не ломает верстку)
- ✅ Шрифты не меньше 14px на мобилках
- ✅ Нет элементов, требующих точного наведения мышью

> 💡 **Совет:** Добавьте сайт на главный экран телефона (Chrome → "Установить приложение") — будет выглядеть как нативное приложение без адресной строки.

### 📚 Что внутри

- **40+ GM команд** с генератором синтаксиса
- **15+ горячих клавиш** для игроков и администраторов
- **8+ координат** ключевых локаций
- **ID предметов** (Адена, свитки, GM вещи)
- **ID скиллов** (Haste, Might, Empower и др.)
- **ID классов** (все профессии до Maestro)
- **Шпаргалки** (уровни GM, формулы заточки)

### 🤔 "Если на сервере другие команды?"

Стандартный синтаксис L2J — один для всех. Но если у вас кастом:

1. Откройте `index.html` в блокноте
2. Найдите массив `db` (там где команды)
3. Добавьте свою команду или измените существующую

```javascript
// Пример добавления кастомной команды
{ 
  id: "my_cmd", 
  cat: "admin_base", 
  type: "admin", 
  cmd: "//mycommand", 
  args: [{n:"arg", v:"value"}], 
  ru: "Моя команда", 
  en: "My command" 
}
```
## 🔧 Расширяемость

Проект спроектирован так, чтобы **легко дополнять его под свои нужды**. Всё, что нужно знать — находится в одном файле `index.html`.

### Как добавить свою команду

```javascript
// Найдите массив db в коде и добавьте объект:
{ 
  id: "my_cmd", 
  cat: "admin_base",           // категория
  type: "admin",               // admin / player / voice / hotkey / info
  cmd: "//mycommand",          // сама команда
  args: [{n:"arg1", v:"value"}], // аргументы (можно пустой массив)
  ru: "Описание на русском", 
  en: "Description in English" 
}
```
Как добавить ID предмета / скилл

```javascript
// Найдите массив dbItems:
{ id: "12345", ru: "Название", en: "Name" }

// Чтобы добавить предмет:
dbItems.push({ id: "99999", ru: "Название", en: "Name" });

// Чтобы добавить скилл:
dbSkills.push({ id: "9999", ru: "Название", en: "Name" });
```

Как добавить координаты

```javascript
// Найдите массив dbLocs:
{ id: "x y z", ru: "Название", en: "Name" }

// Чтобы добавить координаты:
dbLocs.push({ id: "x y z", ru: "Название", en: "Name" });

```

Как добавить категорию
1. Добавьте новую категорию в массив cats в функции renderCategories
2. Добавьте перевод в объект i18n для обоих языков

```javascript
// В i18n.ru и i18n.en:
cat_mycategory: "📁 Моя категория"
```

📁 Структура данных

```javascript
db              // Основная база команд
├── admin_base      // Базовые GM команды
├── admin_spawn     // Спавн и предметы
├── admin_player    // Управление игроками
├── admin_clan      // Кланы и сервер
├── voice           // Голосовые команды (.cfg)
├── player_social   // Чаты и общение
├── player_actions  // Действия и макросы
├── hotkeys         // Горячие клавиши
└── info            // Шпаргалки

dbItems         // ID предметов
dbSkills        // ID скиллов  
dbLocs          // Координаты
dbClasses       // ID классов
```

🤔 "Как это работает без интернета?"

Скачайте index.html на телефон/планшет и откройте в браузере. Всё будет работать локально. Данные (команды, ID) зашиты прямо в файл.
