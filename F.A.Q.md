# ❓ Часто задаваемые вопросы (FAQ)

## 📱 Работа на разных устройствах

### Вопрос: "Куда делась боковая панель с ID на планшете?"

**Ответ:** На планшетах (ширина 768-1024px) боковая панель **скрыта намеренно**, чтобы не загромождать экран. На таком размере три колонки были бы слишком узкими.

| Устройство | Ширина | Боковая панель |
|------------|--------|----------------|
| 🖥️ Десктоп | 1024px+ | ✅ Справа |
| 📟 Планшет | 768-1024px | ❌ Скрыта |
| 📱 Телефон | до 768px | ✅ Внизу страницы |

> **Что делать на планшете?** Используйте поиск — он находит всё мгновенно (ID предметов, команды, координаты).

---

### Вопрос: "На телефоне панель с ID не видно!"

**Ответ:** Она **внизу страницы**. Просто промотайте до конца — там будут вкладки:
- 📦 Items (ID предметов)
- ✨ Skills (ID скиллов)
- 🗺️ Locs (координаты)
- 🛡️ Classes (ID классов)

> **Совет:** Добавьте сайт на главный экран телефона (Chrome → "Установить приложение") — будет выглядеть как нативное приложение.

## 🔧 Расширение и кастомизация

### Вопрос: "Как добавить свою команду?"

**Ответ:** Всё в одном файле `index.html`. Найдите массив `db` и добавьте объект:

```javascript
{ 
  id: "my_cmd",           // уникальный ID
  cat: "admin_base",      // категория
  type: "admin",          // admin / player / voice / hotkey / info
  cmd: "//mycommand",     // сама команда
  args: [{n:"arg1", v:"value"}], // аргументы (можно []),
  ru: "Описание на русском", 
  en: "Description in English" 
}
```
Вопрос: "Как добавить ID предмета / координаты?"
Ответ: Найдите соответствующий массив и добавьте строку:

```javascript
// Добавить предмет
dbItems.push({ id: "99999", ru: "Название", en: "Name" });

// Добавить координаты
dbLocs.push({ id: "x y z", ru: "Название", en: "Name" });

// Добавить скилл
dbSkills.push({ id: "9999", ru: "Название", en: "Name" });

// Добавить класс
dbClasses.push({ id: "999", ru: "Название", en: "Name" });
```

Вопрос: "Сложно добавить новую категорию?"
Ответ: Нет, 3 простых шага:

1. Добавить категорию в массив cats в функции renderCategories:

```javascript
const cats = ["fav", "all", "my_new_cat", ...]
```

2. Добавить перевод в i18n.ru и i18n.en:

```javascript
cat_my_new_cat: "📁 Моя категория"
Добавить команды с cat: "my_new_cat"
```

Вопрос: "А если у меня кастомный сервер с другими командами?"

Ответ: Просто отредактируйте существующие команды или добавьте свои. 
Структура данных простая — легко справится без знания программирования.

## ⭐ Избранное

### ❓ Как работает избранное?
> **Ответ:**  
- Нажмите на ✅ рядом с любой командой — она добавляется в избранное.  
- Перейдите в категорию `⭐ Favorites` — там вы найдёте все сохранённые команды.  
✅ Избранное сохраняется локально (в браузере) и не пропадает после перезагрузки.

---

### ❓ Как перенести избранное на другой компьютер?
> **Ответ:**  
1. Откройте категорию `⭐ Favorites`.  
2. Нажмите 📤 **Экспорт** — скачается JSON-файл.  
3. На новом устройстве:  
   - Перейдите в `⭐ Favorites` → нажмите 📥 **Импорт**  
   - Выберите скачанный файл.  
✅ Все команды перенесутся без потери данных.

---

### ❓ Как очистить всё избранное?
> **Ответ:**  
В категории `⭐ Favorites` нажмите 🗑 **Clear All**, подтвердите действие.  
🗑️ Всё удалится — быстро и безопасно.

### Вопрос: "Как вернуть боковую панель на планшет (перенести вниз, как на телефоне)?"

**Ответ:** По умолчанию на планшетах боковая панель скрыта. Если вам удобнее, чтобы она была внизу (как на телефоне), измените одну строку в CSS.

**Шаг 1:** Откройте файл `index.html` в любом текстовом редакторе (Блокнот, VS Code, Notepad++).

**Шаг 2:** Найдите этот блок кода (примерно строка 70-80):

```css
@media (max-width: 1023px) and (min-width: 768px) { 
    .layout { grid-template-columns: 280px 1fr; } 
}
```
Шаг 3: Замените его на этот:

```css
@media (max-width: 1023px) and (min-width: 768px) { 
    .layout { display: flex; flex-direction: column; } 
}
```
Шаг 4: Сохраните файл и обновите страницу.

Результат: На планшете панель с ID уедет вниз — так же, как на телефоне.

✅ **Альтернативный вариант (быстрый):**

Просто переверните планшет в горизонтальную ориентацию — панель появится справа (так как ширина экрана станет больше 1024px).
